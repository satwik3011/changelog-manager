2024 Out-Of-Cycle Changes

2024 out-of-cycle changes to Graph API and Marketing API.

July 17, 2024

Business Pixel (Dataset) Sharing API Solution

To support use cases for businesses to access a pixel dataset of one business in another business (typically an agency or a partner) for managing ads, analyzing performance or uploading events, the pixel sharing API solution is available.

The following endpoints are now available:

  • POST /{pixel-id}/agencies
  • POST /{pending-request-id}
  • DELETE /{pixel-id}/agencies

At the end of September, the POST /{pixel-id}/shared_accounts API (including previous versions) will not support sharing of pixels with an ad account, if a business account does not have access to both pixel and ad account. Refer to pixel sharing API solution to use POST /{pixel-id}/agencies or POST {ad_account}/agencies to share into a business account, then use POST /{pixel-id}/shared_accounts to link pixel and ad account

March 11, 2024

Ad

Starting March 11th, 2024, we will no longer support providing a bid_amount value on ads. Instead please use the bid_amount value in the ad set as the source of truth. API calls attempting to update the bid_amount for an ad will result in an error.

The following endpoints are affected:

  • GET /{ad-account-id}/ads
  • POST /{ad-account-id}/ads
